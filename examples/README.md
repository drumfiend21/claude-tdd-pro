# Examples — end-to-end walkthroughs

Real, reproducible runs of the plugin. Every artifact is **tool output** (regenerable
deterministically), not hand-written — so these double as living documentation and as
pinned integration fixtures.

## 👉 Start here

- **[dog-walker-marketplace/](dog-walker-marketplace/)** — a non-technical founder
  describes an app in plain English; the plugin interviews them, explains every term
  in business language, and delivers a **complete, world-class, fully-cited** full-stack
  + cloud architecture with a red→green build plan. The single best way to understand
  what claude-tdd-pro does.

Regenerate any example's artifacts with its `regenerate.sh`:

```bash
bash examples/dog-walker-marketplace/regenerate.sh
```

Related: [`docs/DEMO.md`](../docs/DEMO.md) (international marketplace), and the golden
reference designs under [`docs/golden/`](../docs/golden/).
